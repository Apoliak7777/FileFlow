# Changelog

All notable changes to FileFlow will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [7.0] - 2026-06-23

### Added
- Major UI redesign with modern dark theme and improved layout
- Overhauled settings panel with better organization and live preview
- Real statistics in Insights view based on actual usage history
- Improved navigation sidebar for better readability
- Added GitHub link in About section
- Proper LICENSE and CHANGELOG

### Changed
- Updated color palette for a cleaner, more modern look
- Increased sizes and contrast in UI elements for better usability
- Refined file list rows, icons, and buttons
- General code cleanup and comment improvements

### Fixed
- Removed unused/dead code
- Better handling of various edge cases in file operations
- Fixed small UI readability issues

## [6.0] - Previous
- Core features: Live Auto with tray support, powerful Find & Delete, duplicates finder, interactive file management, profiles, rules, command palette, context menu integration.
- Portable single-file executable.
